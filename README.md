## **Steps / Methodology**

1. **Data Loading & Exploration (Python)**

   * Loaded dataset using Pandas
   * Performed exploratory data analysis (EDA) to understand structure, distribution, and correlations

2. **Data Cleaning & Transformation**

   * Handled missing values and duplicates
   * Standardized column names and data formats
   * Created new derived features for analysis

3. **Database Integration (PostgreSQL)**

   * Loaded cleaned data into PostgreSQL
   * Wrote SQL queries to aggregate and summarize data
   * Verified data integrity and correctness

4. **Dashboard Creation (Power BI)**

   * Connected Power BI to PostgreSQL database
   * Created interactive charts, tables, and KPIs
   * Designed dashboard for key insights such as trends, top-performing categories, and customer segments

5. **Reporting & Presentation**

   * Summarized findings in a written report
   * Prepared a professional **PowerPoint presentation** using the GEMMA structure (Goal, Explore, Metrics, Model, Analysis)

## **How to Run the Project**

1. **Python Scripts:**

   * Install dependencies:

     ```bash
     pip install pandas numpy matplotlib seaborn psycopg2
     ```
   * Run the Jupyter notebooks for data cleaning and EDA.

2. **PostgreSQL:**

   * Ensure PostgreSQL server is running
   * Create database and tables using the provided SQL scripts
   * Load the cleaned dataset into the database

3. **Power BI:**

   * Open the Power BI file (`.pbix`)
   * Connect to the PostgreSQL database
   * Refresh the data to view the latest dashboard

4. **Report & Presentation:**

   * Open the report PDF and PPT file for detailed insights and recommendations
